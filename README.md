# UiPath-Outlook-Mail-to-Excel-Automation
This process extracts emails from the "Sample" folder in the Outlook inbox and exports the email data to Excel. Attachments, if present, are automatically saved to a designated folder. Upon completion, a notification email is sent automatically.

## 概要
UiPathを使用し、Outlookのメール情報をExcelへ自動転記する業務自動化ツールです。  
特定フォルダ内のメールを対象に、添付ファイルの保存および処理完了通知まで自動化します。

---

## Features
- Extract emails from Outlook folder ("Sample")
- Export email data to Excel
- Save attachments to a specified folder
- Send completion notification email after processing

---

## 機能
- Outlook受信トレイ内「Sample」フォルダのメールを取得
- メール情報をExcelへ自動転記
- 添付ファイルを指定フォルダへ保存
- 処理完了後に通知メールを自動送信

---

## Process Flow
1. Access Outlook folder ("Sample")
2. Retrieve email data
3. Export data to Excel
4. Save attachments (if exists)
5. Send completion notification email

---

## 処理フロー
1. Outlook「Sample」フォルダへアクセス
2. メール情報を取得
3. Excelへ転記
4. 添付ファイルを保存（存在する場合）
5. 完了通知メールを送信
